# v1.0 系统配置域角色 / 权限 / 策略组合矩阵

## 1. 文档目的

本文档用于固定 `JOB-090` 在系统配置域中需要重点覆盖的组合矩阵。

## 2. 适用范围

覆盖页面 / 链路：

- 用户
- 角色
- 权限
- 治理策略
- 关键动作权限判定

## 3. 测试矩阵

| 用户 / 角色 | 预期能力 | 禁止能力 | 主要验证点 |
| --- | --- | --- | --- |
| `viewer_user / dict_viewer` | 查看页面、帮助、摘要 | 写词典建设数据、审核、发布 | 只读权限 |
| `editor_user / dict_editor` | 新增词条、导入、提交审核 | 审核决策、发布 | 编辑能力 |
| `reviewer_user / dict_reviewer` | 审核决策 | build / publish | 审核能力 |
| `reviewer_user2 / dict_reviewer` | 第二审核人场景 | build / publish | 双审核场景 |
| `publisher_user / dict_publisher` | build、提审、发布 | 审核决策 | 发布能力 |
| `operator_user / dict_operator` | 灰度、回滚、节点控制 | 词典建设写入 | 运维能力 |
| `console_user / dict_admin` | 全部功能 | 无 | 管理员兜底 |

## 4. 治理策略组合

| 策略 | 场景 | 预期 |
| --- | --- | --- |
| 提交人与审核人分离 | 同人提交并审核 | 被阻断或有明确约束 |
| 审核人与发布执行分离 | 同人审核并发布 | 按策略生效 |
| 高风险双人审批 | 高风险 release | 需要双审核人 |
| 默认审批人数 | 普通 release | 满足默认门槛即可 |

## 5. 当前用途

本文档当前作为：

- `JOB-090` 第一轮系统配置域组合矩阵
- `JOB-035` 落地时用户、角色、权限同步调整的复核依据
- 后续测试 Codex 做 RBAC 与治理策略交叉验证时的基线
