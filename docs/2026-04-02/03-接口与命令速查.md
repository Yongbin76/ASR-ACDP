## 1. 运行入口

- `npm run check:env`

## 6. 当前路由与接口关系

- `/admin`
  - MVP 页面对接 `GET /api/admin/*`
- `/console`
  - 单页应用，对接 `GET/POST /api/console/*`
- runtime 节点
  - 不走 `/admin` 角色头，而走 Bearer Token 控制链路

## 7. 推荐排查顺序

### 7.1 本地启动失败

1. `npm run check:env`
2. `npm run setup:prototype`
3. `npm run smoke:console`
4. `npm run smoke:runtime`
5. `npm run smoke:admin`

### 7.2 runtime-control 异常

1. `npm run check:control-config`
2. `npm run local:minio:status`
3. `npm run verify:runtime-control -- --artifact-store-mode=configured`
4. `npm run verify:runtime-control -- --artifact-store-mode=file`

### 7.3 集群部署异常

1. `npm run check:k8s-target`
2. 检查 `k8s/artifact-store-secret.example.yaml`
3. 检查 `runtime-deployment.yaml` / `admin-deployment.yaml` 的 Secret 注入
