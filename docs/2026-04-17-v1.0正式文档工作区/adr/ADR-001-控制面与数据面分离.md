# ADR-001 控制面与数据面分离

- 文档状态：active
- 适用版本：v1.0
- 文档类型：adr
- 所属工作区：2026-04-17-v1.0正式文档工作区
- 最后更新时间：2026-04-17
- 决策状态：accepted

## 1. 决策

admin 负责控制面，runtime 负责数据面。

## 2. 理由

1. 管理后台、词典建设、审核、发布和节点治理属于控制面职责。
2. 文本纠错、snapshot 加载、灰度分流和本地统计属于 runtime 数据面职责。
3. 这两类职责如果混在同一套对外合同里，后续多 runtime 与发布治理会持续混乱。

## 3. 结果

1. `/api/console/*` 和 `/api/admin/*` 归属 admin。
2. `/api/runtime/*` 和 `/ws/runtime/*` 归属 runtime。
3. runtime 通过 runtime-control 接口与 admin 同步。
