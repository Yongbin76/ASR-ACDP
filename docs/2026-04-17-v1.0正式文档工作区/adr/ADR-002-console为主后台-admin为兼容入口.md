# ADR-002 console 为主后台，admin 为兼容入口

- 文档状态：active
- 适用版本：v1.0
- 文档类型：adr
- 所属工作区：2026-04-17-v1.0正式文档工作区
- 最后更新时间：2026-04-17
- 决策状态：accepted

## 1. 决策

`/console` 是当前版本的正式后台主入口，`/admin` 只作为兼容入口保留。

## 2. 理由

1. 当前正式信息架构、目录树、帮助体系和页面职责都已在 `/console` 收口。
2. `/admin` 仍保留部分原型和兼容页面，不适合继续承担主流程设计。

## 3. 结果

1. 后续正式验收以 `/console` 为准。
2. 后续衍生开发优先更新 `/console` 对应文档和接口。
3. `/admin` 不再作为新的主流程设计基线。
