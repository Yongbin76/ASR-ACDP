# 数据字典索引

- 文档状态：active
- 适用版本：v1.0
- 文档类型：data_dictionary_index
- 所属工作区：2026-04-17-v1.0正式文档工作区
- 最后更新时间：2026-04-17
- 责任对象：SQLite 表结构、字段、类型、约束与案例

## 1. 作用

本目录用于把当前 SQLite 控制库做成“可开发、可修改、可回归”的详细数据字典。

它不是简单字段注释，而要说明：

1. 字段类型。
2. 可空性。
3. 默认值。
4. 主键、唯一键、外键、索引。
5. 业务含义。
6. 写入路径与读取路径。
7. 改字段时要同步哪些代码和测试。
