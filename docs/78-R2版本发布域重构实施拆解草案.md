## 2. R2 目标

**重建版本发布域，使 release 生命周期、发布审核、灰度、正式发布、回滚、发布后风险彻底分层。**

## 6. 收尾标准

`R2` 只有在以下条件全部满足时才能关单：

1. 版本列表与详情页生命周期语义清晰
2. 发布审核、灰度、正式发布、回滚、发布后风险不再混读
3. `built` 后不再被主数据 gate 反向阻断
4. 发布域帮助与流程说明齐全
5. 发布域相关回归通过
