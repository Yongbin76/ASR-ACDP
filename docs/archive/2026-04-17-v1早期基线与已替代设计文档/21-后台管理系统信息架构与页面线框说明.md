## 1. 文档目的

本文件把新后台的信息架构和关键页面线框进一步落地，目标是让开发在编码前对页面骨架和跳转关系没有歧义。

本文件承接：

- [16-后台管理系统页面与用户流程原型](./16-后台管理系统页面与用户流程原型.md)
- [20-后台管理系统视觉风格与交互基线](./20-后台管理系统视觉风格与交互基线.md)

## 2. 页面结构总览

新后台采用：

- 左侧一级导航
- 顶部上下文栏
- 中间独立页面
- 详情页内部使用分区或轻量标签

不采用：

- 一个页面下的多功能 sheet/tab 主结构

## 3. 一级导航结构

## 5. 页面线框说明

## 5.1 工作台页

页面骨架：

1. 页面标题区
2. 核心摘要卡区
3. 最近待处理事项区
4. 最近发布与验证区
5. 最近导入区

摘要卡建议包括：

- 词条总数
- 错误词总数
- 待审核
- 当前 stable 版本
- 当前 canary 版本
- 最近验证失败数

## 5.2 词条列表页

页面骨架：

1. 页面标题 + 新建按钮
2. 筛选区
3. 列表表格
4. 分页区

筛选区字段建议：

- 搜索词
- 类别
- 状态
- 来源
- 风险等级
- 排序方式

当前已落地补充：

- 页面顶部先展示摘要卡，再进入筛选区和列表区
- 右侧保留独立“新建词条”工作区，不与列表混排混用
- 列表底部已补统一分页栏，翻页时保留当前筛选与排序条件
- 列表区已支持“当前页勾选”与“当前筛选结果”两种批量送审 / 批量停用模式，仍不支持跨页手工累积选择
- 列表区已补“导出当前筛选”入口，直接输出 CSV 附件

表格列建议：

- 标准词
- 类别
- 来源
- 状态
- 错误词数
- 风险等级
- 更新时间
- 发布状态
- 操作

## 5.3 词条详情页

页面骨架：

1. 顶部摘要卡
2. 区块导航
3. 分区内容

顶部摘要卡建议展示：

- 标准词
- 类别
- 状态
- 风险等级
- 来源
- 当前发布状态
- 快捷动作按钮

区块顺序：

1. 基本信息
2. 错误词/别名
3. 规则
4. 拼音治理
5. 来源信息
6. 审核历史
7. 发布历史
8. 验证表现

## 5.4 批量导入首页

页面骨架：

1. 页面标题
2. 导入状态摘要卡
2. 模板入口卡片区
3. 最近导入批次列表

当前已落地补充：

- 批次列表已支持按批次状态、来源类型筛选
- 已导入批次优先展示导入结果摘要，待确认批次展示预览统计
- 列表底部已补统一分页栏，翻页时保留当前筛选条件

模板卡片必须包含：

- 模板名称
- 适用场景
- 支持格式
- 模板下载
- 示例文件下载
- 去创建导入按钮

## 5.5 导入批次详情页

页面骨架：

1. 批次摘要卡
2. 上传文件信息
3. 预览汇总
4. 预览行 / 导入行筛选区
5. 行级明细表

当前已落地补充：

- 不采用页内主标签，而是统一纵向信息流
- 详情页已支持按行状态、处理决策筛选
- 已导入批次会把顶部摘要卡切换为“新增词条 / 更新词条 / 新增错误词 / 错误数”
- 行级明细表已补统一分页栏，翻页时保留当前筛选条件

## 5.6 内容审核页

页面骨架：

1. 页面标题
2. 任务摘要卡
2. 任务筛选区
3. 审核任务摘要条
4. 审核任务卡片流
5. 详情独立页

筛选建议：

- 任务类型
- 目标类型
- 状态
- 提交人
- 审核人

## 5.7 版本列表页

页面骨架：

1. 页面标题 + 构建 release 按钮
2. 发布状态摘要
3. release 列表

当前已落地补充：

- 发布列表已支持按 `status` 筛选
- built/canary/published 摘要卡直接放在列表页顶部
- 列表底部已补统一分页栏，翻页时保留当前筛选条件

列表列建议：

- 版本号
- 版本状态
- 审批状态
- 流量状态
- term 数量
- 创建时间
- 发布时间
- 操作

## 5.8 发布详情页

页面骨架：

1. release 摘要卡
2. 当前动作
3. 版本状态 / 发布审批 / 流量状态
4. 发布确认 / 发布后风险
5. 节点收敛与验证证据
6. 变更词条

说明：

- `built` 是候选快照状态，不受 gate 阻断
- gate 只作为 `built/canary` 升级前置条件
- `published` 后若再出现异常，页面统一归为“发布后风险”，不再继续混叫“门禁阻断”

## 5.9 样本列表页

页面骨架：

1. 页面标题
2. 筛选区
3. 样本表格
4. 分页区

列建议：

- 样本 ID
- 说明
- 来源
- 启用状态
- 期望标准词数
- 最近更新时间
- 操作

当前已落地补充：

- 列表页已支持按关键字、来源类型、启用状态筛选
- 右侧保留独立“新增样本”工作区，方便一边查一边补录

## 5.10 系统配置页

页面骨架：

1. 系统配置首页
2. 基础配置区
3. 节点治理区
4. 运行验证入口
5. 用户 / 角色 / 权限 / 治理策略入口

说明：

- 用户、角色、权限、治理策略统一从配置种子读取并通过管理页维护
- 节点备案、运行节点和运行验证归到同一域，不再散落在业务主导航中
- 右侧已补“文件批量导入样本”工作区，主入口采用模板下载 + 文件上传 + 预览确认闭环
- JSON 粘贴导入仍保留为兼容调试入口，不作为正式业务导入主路径
- 列表底部已补统一分页栏，翻页时保留当前筛选条件
- 列表区已支持“当前页勾选”与“当前筛选结果”两种批量停用模式，仍不支持跨页手工累积选择
- 列表区已补“导出当前筛选”入口，直接输出 CSV 附件

## 5.10 样本详情页

页面骨架：

1. 顶部摘要卡
2. 样本正文
3. 期望标准词
4. 关联词条
5. 最近验证结果

## 6. 页面跳转关系

必须保证以下直达关系：

- 工作台 -> 词条
- 工作台 -> 批量导入
- 工作台 -> 内容审核
- 工作台 -> 版本列表
- 词条列表 -> 词条详情
- 词条详情 -> 关联验证样本
- 批量导入 -> 导入批次详情
- 导入批次详情 -> 导入产生的词条
- 内容审核 -> 目标对象详情
- 发布详情 -> 失败样本或问题词条
- 样本详情 -> 关联词条

## 7. 编码前结论

开发可直接按本文件确定：

- 页面级路由
- 一级导航结构
- 各页面骨架布局
- 详情页分区结构
- 必要的跨页面跳转关系
