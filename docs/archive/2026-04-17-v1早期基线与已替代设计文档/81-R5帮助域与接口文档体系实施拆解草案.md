### 3.1 页面帮助

- 工作台
- 主数据域页面
- 内容审核页
- 版本发布页
- 运行治理页
- 系统配置页

### 3.2 流程帮助

- 主数据录入到发布
- 内容审核
- 发布 / 灰度 / 正式发布 / 回滚
- 节点接入与运行验证

### 3.3 运维与排障帮助

- admin 启动与部署
- runtime 启动与部署
- 多 runtime 实例部署
- 节点注册排查
- 发布失败排查
- 发布后风险排查

### 3.4 接口帮助

- runtime 对外 HTTP 接口
- runtime 对外 WebSocket 接口
- admin / console 管理接口帮助
