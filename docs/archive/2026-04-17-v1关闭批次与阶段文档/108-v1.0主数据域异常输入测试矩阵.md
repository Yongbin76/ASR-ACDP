# v1.0 主数据域异常输入测试矩阵

> 口径说明
>
> 本文档标题中的 `主数据域` 为旧测试口径。
>
> 当前应理解为：
>
> - `词典建设` 录入、导入与审核前准入异常
> - 与其直接相关的 `验证与回流` 配置消费异常

## 1. 文档目的

本文档用于固定 `JOB-090` 在主数据域中需要重点覆盖的异常输入矩阵。

## 2. 适用范围

覆盖页面 / 链路：

- 词条新增
- 词条编辑
- 结构化词条导入
- 错误词补录
- validation 样本写入

## 3. 测试矩阵

| 类别 | 样本类型 | 预期 | 主要验证点 |
| --- | --- | --- | --- |
| 准入阻断 | 单字标准词无错误词 / 含单字错误词 | 阻断 | 单字标准词特例准入拦截 |
| 准入阻断 | 栏目词 / 说明词 | 阻断 | 非词条类内容拦截 |
| 准入阻断 | `canonicalText` 命中 alias | 阻断 | 冲突 trace |
| 准入阻断 | 非法业务属性 | 阻断 | 业务属性校验 |
| 准入阻断 | 拼音字段非法格式 | 阻断 | 拼音字段校验 |
| 风险提示 | 路名特征弱匹配 | warning | `poi_road` 轻量校验 |
| 风险提示 | 政府部门特征弱匹配 | warning | `gov_term` 轻量校验 |
| 风险提示 | 发布后风险关联样本 | warning | validation 样本提醒 |
| 导入阻断 | CSV 缺必填列 | 阻断 | 模板校验 |
| 导入阻断 | 批次内错误行 | 阻断批次确认 | blocked 行不可确认 |
| 补录阻断 | 标准词不存在 | 阻断 | alias 补录前置 |
| 样本阻断 | validation 缺 text / expected | 阻断 | 样本基本字段 |

## 4. 当前用途

本文档当前作为：

- `JOB-090` 第一轮主数据域异常输入测试矩阵
- 后续测试 Codex 补主数据域异常输入场景时的直接基线
