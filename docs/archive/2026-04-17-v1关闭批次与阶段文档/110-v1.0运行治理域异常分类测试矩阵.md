# v1.0 运行治理域异常分类测试矩阵

## 1. 文档目的

本文档用于固定 `JOB-090` 在运行治理域中需要重点覆盖的异常分类矩阵。

## 2. 适用范围

覆盖页面 / 链路：

- 节点备案
- 运行节点列表
- 运行节点详情
- 运行验证

## 3. 测试矩阵

| 类别 | 场景 | 预期 | 主要验证点 |
| --- | --- | --- | --- |
| 备案 | 已备案未注册 | 主列表可见，状态明确 | 备案节点是主对象 |
| 备案 | 未备案但上报 | 不进入主列表，进入异常接入 | 未备案节点处理 |
| 注册 | secret 不匹配 | 注册失败 | 注册错误分类 |
| 注册 | token 缺失 | 注册失败 | 鉴权错误分类 |
| 在线 | 已注册但无心跳 | 标记未心跳/离线 | 实时状态口径 |
| 在线 | 离线后恢复 | 当前在线，历史异常保留 | 实时/历史分离 |
| 对齐 | 目标版本不一致 | 标记未对齐 | 目标状态与当前状态分离 |
| 对齐 | 最近 apply 失败 | 最近动作失败，但不覆盖当前在线态 | 最近动作结果分层 |
| 运行验证 | cluster_current 验证成功 | 验证结果可读 | 运行验证入口 |
| 运行验证 | 节点异常存在时 | 页面有明确说明 | 运行治理解释性 |

## 4. 当前用途

本文档当前作为：

- `JOB-090` 第一轮运行治理域异常分类矩阵
- 后续测试 Codex 补运行治理域细分场景时的直接基线
