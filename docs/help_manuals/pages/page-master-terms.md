# 页面手册：词典记录

## 1. 页面目标

`词典记录` 页用于维护`词典建设`中的词典记录本身。

它负责：

- 新建词典记录
- 编辑基础信息
- 编辑规则
- 编辑拼音
- 提交词典审核

它不直接负责：

- 改当前 release
- 触发正式发布

## 2. 页面主对象

- 词典记录

## 3. 主要区块

### 3.1 新建词典记录

用于快速录入：

- 业务属性
- 标准词
- 错误词 / 别名
- 优先级
- 风险等级
- 替换模式
- 来源类型

补充规则：

- 当标准词是单字时，必须填写至少一个错误词 / 别名
- 只要错误词 / 别名里存在任意单字，该词条就会被统一准入阻断
- 单字标准词只有在“错误词均为非单字”时，才允许继续录入

### 3.2 词典记录列表

用于：

- 查看词条当前状态
- 查看审核状态
- 进入详情
- 提交审核

补充说明：

- 当词条已经审核通过，且通过后没有新的内容改动时，页面不会再显示“提交审核”
- 只有在审核通过后发生了新改动，或当前词条仍未形成有效审核结果时，才会再次出现“提交审核”
- 已审核通过的拼音候选不会再次显示“提交候选审核”；即使绕过页面直接调用接口，后端也会拒绝重复提交
- 当前页已把“批量动作层”调整到“筛选层”之前
- 当前页的筛选动作现在只刷新结果层，不再整页重刷
- 当前页的上一页 / 下一页翻页动作现在也只刷新结果层，不再整页重刷

当前“词条状态”筛选包含：

- 草稿
- 待审核
- 已通过
- 已停用

## 4. 统计口径

- 顶部卡片：显示词典记录全量口径，不随筛选和分页变化
- 顶部卡片：总量固定显示；值为 0 的次级状态卡片默认不展示
- 结果区摘要：显示当前筛选结果口径
- 表格：只显示当前页数据
- 分页：固定放在结果列表区块底部

## 4.1 当前页分层顺序

当前页的主工作区顺序为：

1. 批量动作与导出
2. 筛选条件
3. 结果列表（含分页）

## 5. 当前页下一步

词条录入完成后，下一步通常是：

1. 保存草稿
2. 进入词典记录详情补规则 / 拼音
3. 提交词典审核
4. 审核通过后等待参与下一次 build
