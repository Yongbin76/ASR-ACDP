# 页面手册：版本列表

## 1. 页面目标

`版本列表` 页用于查看所有 release 的生命周期状态，并从列表进入详情页继续处理发布审核、灰度、正式发布、回滚和发布后风险。

## 2. 页面主对象

- release 候选快照

## 3. 主要区块

### 3.1 状态概览

列表固定拆成：

- 版本状态
- 发布审核
- 流量状态
- 版本校验 / 发布后风险

### 3.2 构建新版本

用于从当前可发布输入池创建新的 `built` release。

## 4. 统计口径

- 顶部卡片：显示 release 全量口径，不随视图切换、筛选和分页变化
- 顶部卡片：总量固定显示；值为 0 的次级状态卡片默认不展示
- 结果区摘要：显示当前工作视图和当前筛选结果口径
- 表格：只显示当前页版本
- 分页：固定放在当前工作视图结果区块底部

## 5. 主要按钮

### 4.1 `查看详情`

- 作用：进入该 release 详情页
- 下一步：继续处理审核、灰度、正式发布、回滚或风险观察

### 4.2 `创建并构建`

- 作用：从当前输入池构建新的候选快照
- 输出：新建一个 `built` release
