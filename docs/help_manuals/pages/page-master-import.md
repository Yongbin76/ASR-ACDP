# 页面手册：批量导入

## 1. 页面目标

`批量导入` 页用于批量录入词典建设数据。

当前主要包括：

- 结构化词条导入
- 错误词补录

## 2. 核心原则

- 导入写入的是词典建设数据
- 不是直接修改某个已存在 release

## 3. 标准流程

1. 选择模板
2. 下载模板或示例
3. 上传文件
4. 查看预览
5. 确认导入
6. 回看影响词条和审核任务
7. 如果生成了大量词条审核任务，进入“本批词条审核”视图集中处理

补充说明：

- `取消批次` 只允许在预览阶段使用
- 已导入批次不能再取消
- 首页主任务卡统一只保留两个按钮：`查看字段说明`、`下载模板及示例`
- `结构化词条导入模板` 与 `错误词批量补录模板` 的详情页“参考与下载”也统一为：`查看字段说明`、`下载模板及示例`

## 4. 导入后审核闭环

- 导入只负责把词典建设数据写入词典记录
- 导入不会直接让数据进入 release
- 导入完成后，新增或更新词典记录会生成 `term_review`
- 建议从导入详情页进入“前往词条审核本批任务”，避免在宽审核列表里逐条查找

## 5. 统计口径

- 顶部卡片：显示导入批次全量口径，不随筛选和分页变化
- 顶部卡片：总量固定显示；值为 0 的次级状态卡片默认不展示
- 批次结果区摘要：显示当前筛选结果口径
- 表格：只显示当前页批次
- 分页：固定放在批次结果区块底部

## 5.1 当前页分层顺序

当前页的主工作区顺序为：

1. 创建导入批次
2. 最近导入批次

补充说明：

- `创建导入批次` 当前采用左右分栏：
  - 左侧约 70%：导入表单
  - 右侧约 30%：`结构化词条导入模板`、`错误词批量补录模板`、`导入链路说明`，上下排列
- `最近导入批次` 的筛选条件区当前为一行结构：
  - 2 个选择条件
  - 2 个动作按钮
