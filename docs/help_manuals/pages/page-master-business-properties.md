# 页面手册：基础配置

## 1. 页面目标

`基础配置` 页用于统一维护`词典建设`中的基础字典配置。

当前页包含两块：

- 业务属性配置
- 来源类型配置

当前布局补充：

- 左侧主区当前顺序为：`配置信息 -> 业务属性配置 -> 来源类型配置`
- 右侧侧栏当前承接：`新增/编辑业务属性`、`新增/编辑来源类型`

## 2. 业务属性配置

- 编码
- 中文名称
- 说明
- 启用 / 停用

它会同时影响：

- 词典记录页的业务属性选项
- 批量导入页的业务属性选项

## 3. 来源类型配置

- 编码
- 中文名称
- 说明
- 适用域
- 允许录入方式
- 启用 / 停用

## 4. 注意事项

- 当前只允许新增、编辑、启停
- 已被引用的业务属性不应物理删除
- 已被引用的来源类型不应物理删除
- 停用后会从前台可选项中隐藏
