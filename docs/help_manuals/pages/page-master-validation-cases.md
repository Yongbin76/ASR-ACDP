# 页面手册：验证样本

## 1. 页面目标

`验证样本` 页属于`验证与回流`域，不再属于`词典建设`域。

它负责维护：

- validation case
- 外部回流样本
- 样本导入与停用

## 2. 核心原则

- 样本用于验证词典效果，不是词典内容本体
- 样本不是 release 页面
- 样本用于后续 build 后的版本校验

## 3. 主要动作

- 创建样本
- 模板导入样本
- JSON 调试导入
- 停用样本
- 查看关联词条

## 4. 统计口径

- 顶部卡片：显示验证样本全量口径，不随筛选和分页变化
- 顶部卡片：总量固定显示；值为 0 的次级状态卡片默认不展示
- 结果区摘要：显示当前筛选结果口径
- 表格：只显示当前页样本
- 分页：固定放在样本结果列表区块底部
