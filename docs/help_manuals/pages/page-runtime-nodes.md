# 页面手册：运行节点

## 1. 页面目标

`运行节点` 页用于以备案节点为主对象，查看注册状态、实时状态、目标版本、最近动作和历史异常。

## 2. 适用角色

- 运维操作人
- 发布人
- 管理员

## 3. 页面主对象

- 已备案节点

补充：

- 未备案 runtime 不进入主列表
- 未备案 runtime 只作为异常接入事件提示

## 4. 页面区块

### 4.1 顶部摘要

用于快速查看：

- 备案节点数
- 已禁用备案数
- 未注册节点数
- 在线节点数
- 离线节点数
- 当前异常节点数
- 未备案接入事件数

统计口径：

- 顶部卡片：显示运行节点全量口径，不随筛选和分页变化
- 顶部卡片：备案节点总量固定显示；值为 0 的次级状态卡片默认不展示
- 结果区摘要：显示当前筛选结果口径
- 表格：只显示当前页备案节点
- 分页：固定放在节点状态结果区块底部

### 4.2 运行节点列表

列表按以下顺序展示：

1. 备案状态
2. 注册状态
3. 实时状态
4. 目标状态
5. 最近动作
6. 历史异常

### 4.3 当前下发目标版本

用于查看：

- 当前目标 release
- 已对齐节点数
- 待收敛节点数
- 失败节点数

并可执行：

- 手动刷新
- 开关自动刷新
- 重新下发当前目标版本

## 5. 主要按钮

### 5.1 `查看详情`

- 作用：进入该备案节点的详情页
- 下一步：查看注册、实时状态、目标版本、最近动作和历史异常

### 5.2 `重新下发当前目标版本`

- 作用：把当前目标版本重新下发给 runtime
- 前置条件：具备发布下发权限
- 下一步：观察节点收敛

## 6. 状态解释

### 6.1 备案状态

- `备案已启用`
- `备案已禁用`

### 6.2 注册状态

- `已注册`
- `未注册`

### 6.3 实时状态

- `在线`
- `离线`
- `未注册`

### 6.4 历史异常

历史异常仅用于追溯，不代表当前节点仍有故障。
