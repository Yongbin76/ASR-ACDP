## 4. 当前主要缺口

## 4.1 可立即补齐的缺口

1. 域级缺口矩阵需要固定到真源或测试索引
2. 测试执行顺序和报告格式需要固定
3. 缺陷回流模板需要固定
4. `JOB-006 / JOB-009 / JOB-030` 的 runbook 需要固定

## 4.2 依赖真实环境的缺口

1. 真实集群部署验证
2. 真实宿主机吞吐验证
3. 真实发布 / 回滚演练留档

## 4.3 需要后续讨论的缺口

1. 是否升级到更强的像素级视觉回归
2. 是否把当前覆盖率门槛继续提升到更高基线

## 5. 建议测试优先级

### P0

- 真源 / 文档 / 帮助一致性
- 主流程工作流
- 发布域与运行治理域

### P1

- 系统配置域矩阵
- 帮助域入口一致性
- 发布后风险细分场景

### P2

- 扩展性能模型
- 更细的历史数据污染场景
- 视觉回归（当前已落最小哈希对比方案）
