# v1.0 发布前本地前置检查清单

## 1. 文档目的

本文档用于把进入 `JOB-030` 之前，本地侧必须先确认的项目固定下来。

目标是：

- 在真实部署与真实吞吐条件到位之前，先确认本地准备是否齐全
- 避免进入 `JOB-030` 时仍缺文档、脚本、模板或基础回归能力
- 给 Codex、测试和发布负责人一份统一的本地前置检查表

## 2. 检查范围

本清单只覆盖**本地前置条件**，不代替：

- `JOB-006` 真实部署验证
- `JOB-009` 真实并发与吞吐验证

## 3. 必须项

- [ ] `npm run pm:check` 通过

本文档当前用于：

- 作为进入 `JOB-030` 之前的本地检查依据
